# 基于原生HTML+CSS+JS的起始页/导航页


## 特点
- 精简代码
- 侧边栏整合
- 和风天气API
- 一言API
- 今日诗词API
- 深色模式（跟随系统）
- 增加搜索智能提示

## 引用外链
 - 和风天气API https://www.qweather.com
 - 一言API https://v1.hitokoto.cn
 - 今日诗词API https://www.jinrishici.com
 - 字体 https://fonts.googleapis.com
 - 图标引用阿里图标 https://www.iconfont.cn
