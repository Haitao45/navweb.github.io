# HTML+CSS+JS的极简起始页/静态页/导航页

## 与原版的改动

- 精简代码
- CSS: 去除无用CSS，精简合并，优化侧边栏动画
- 文件结构更加整齐（重定位iconfont，css，js文件夹到static下）
- 更新和风天气API
- 大量优化

## 增加内容

- 添加底部仿 V2EX 延迟检测（纯js，可以一定程度上检测延迟高低，但准确性无法保证）

- 增加一言api

- 增加今日诗词api

- 增加深色模式（跟随系统）
